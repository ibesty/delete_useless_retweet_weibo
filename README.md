# delete_useless_retweet_weibo
删除无用的转发微博（原博已被删除/不可见 等）

使用方法：
1. 将下面的蓝色链接拖到书签栏
2. 打开微博网页版，进入到个人主页（点击你自己头像进入到那个页面） -> 全部微博
3. 点击书签栏保存的书签

之后页面会自己滚动，找到无用的转发微博，并且删除掉
当删除的微博过多时可能会被微博限制，需要稍微等一会之后再次点击书签

## 如果出现异常现象刷新浏览器即可 提issue以便我作修改
